# Enter the Rabbit Hole 🐇

A surreal, interactive narrative experience with **16 unique endings**, hidden secrets, and a mystery that only 2% of players discover.

**[▶️ PLAY NOW](https://kynaruniverse.github.io/enter-the-rabbit-hole/)**

## Features
- **16 branching endings** — choose wisely, or spiral endlessly
- **Daily randomization** — something new appears each day
- **Secret ending** — hidden behind an obscure sequence (hint: ↑↑↓↓←→←→ba)
- **NPC encounters** — mysterious characters that only appear if you trigger them
- **Share cards** — beautiful shareable result images with unique ending codes
- **No account needed** — works offline, saves locally
- **Mobile-friendly** — works on your phone

## How to Play
1. Choose a portal (red, blue, or mysterious gap)
2. Navigate through 7 layers of surreal choices
3. Each path leads somewhere different
4. Find all 16 endings — or just find the secret

⚠️ **Warning:** There is no going back.

## Try to Find
- The ending where you're always too late
- The path that circles back
- The secret only 1 in 50 players discover
- The konami code hidden in the experience

## Built With
- **Pure HTML/CSS/JavaScript** (no frameworks)
- **Canvas-based result cards** for shareability
- **Supabase** for global statistics
- **Local storage** for persistent playthroughs
- **Service Worker** for offline support
